详细描述

科大讯飞TTS芯片 XFS5152CE BSP程序

作者
高明飞
日期
2015-09-24

本模块封装好了XFS5152CE的接口，使用时需要先配置好底层通信驱动， 之后直接调用相关函数即可。
需要先调用初始化函数。

注解
通信方式可配置为UART、SPI、I2C，此BSP程序中不涉及具体实现方式。

宏定义说明

#define XFS5152CE_BUSY   P42

返回RDY引脚电平

返回值
H芯片正忙
L芯片空闲
注解
程序中可直接调用此宏来判断是否发送新数据
#define XFS5152CE_READ_BYTE (   XFSData)    I2CReadByte(0x40,XFSData)

读取一个字节数据宏定义

参数
XFSData读取到的数据结果储存位置指针
#define XFS5152CE_SEND_BYTES (   XFSData,
  XFSCount 
)    I2CWriteMultiBytes(0x40,XFSCount,XFSData)

发送一段数据宏定义

参数
XFSData待发送数据指针
XFSCount数据个数
#define XFS_BUFFER_SIZE   50

数据缓存区大小，一次能发送的中文个数最多为 XFS_BUFFER_SIZE / 2 个

警告
-XFS_BUFFER_SIZE要小于100，否则程序中某些地方会发生溢出;
-为支持文本控制标记，XFS_BUFFER_SIZE需要大于16.
